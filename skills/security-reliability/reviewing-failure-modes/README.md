# Failure Mode Review — plain-English guide

## What this skill is for

Reviews how systems fail under dependency outages, partial writes, retries, and degraded states. Use when services coordinate remote calls, queues, or critical transactions.

## In simple terms

This skill helps with failure mode review work so you do not have to explain the same rules or workflow every time.

## When you would use it

- failure mode
- partial outage
- degraded mode
- retry storm
- fallback

## Example things you could ask for

- Use the reviewing-failure-modes skill to help with this task.
- Reviews how systems fail under dependency outages, partial writes, retries, and degraded states. Help me when services coordinate remote calls, queues, or critical transactions.
