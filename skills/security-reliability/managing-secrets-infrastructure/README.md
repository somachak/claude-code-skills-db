# Managing Secrets Infrastructure

**Category:** security-reliability · **Priority:** supporting · **Audience:** backend, full-stack

Designs and audits secrets infrastructure at scale: HashiCorp Vault, AWS Secrets Manager, Azure Key Vault, GCP Secret Manager. Use when moving beyond .env files to production secrets management — dynamic credentials, automatic rotation, audit logging, and CI/CD integration without long-lived tokens. Covers AppRole auth, Kubernetes pod identities, dynamic database credentials, certificate lifecycle, and 15-minute emergency rotation procedures. Different from auditing-secrets-and-config which audits existing code-level practices.
