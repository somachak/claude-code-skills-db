# Audit Context Builder

**Category:** security-reliability · **Priority:** supporting · **Audience:** backend, full-stack

Builds ultra-granular architectural context through systematic line-by-line code analysis before vulnerability hunting. Use before security audits, threat modeling sessions, or any deep code review where superficial understanding risks missing subtle bugs. Five-phase: orientation, micro-analysis per function, global system reconstruction, stability rules, integration. Do NOT use for generating vulnerability findings — this is the pre-audit context phase only.
