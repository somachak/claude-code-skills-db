# React Animation with Motion.dev

**Category:** frontend · **Priority:** supporting · **Audience:** full-stack, frontend

Covers the animation pattern decision tree (entrance, gesture, scroll, layout), spring physics tuning, stagger orchestration, prefers-reduced-motion accessibility, and performance validation. Use when adding page transitions, scroll reveals, hover interactions, drag gestures, or layout animations.
