# React Rendering Debugger — plain-English guide

## What this skill is for

Diagnoses unnecessary renders, stale closures, memoization mistakes, and state propagation issues in React applications. Use when UI feels slow, renders are noisy, or component updates are confusing.

## In simple terms

This skill helps with react rendering debugger work so you do not have to explain the same rules or workflow every time.

- `stale closure` — A bug where code keeps using an old value because it captured earlier state.
