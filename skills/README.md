# Skills Library

Installable Claude Code skills grouped by category. Every skill is a directory with a `SKILL.md` (what Claude reads) and a `README.md` (what humans read).

## Categories

- [`ai-productivity/`](./ai-productivity/) — 11 skills. Extracting reusable skills, reviewing PRs with Claude, planning multi-agent work, building RAG-ready docs, and cataloging patterns.
- [`backend/`](./backend/) — 11 skills. Node.js, FastAPI, and Django patterns for APIs, auth, jobs, concurrency, webhooks, event-driven systems, and backend documentation.
- [`data/`](./data/) — 7 skills. Schema design, SQL review, migrations, search/indexing, analytics instrumentation, and ETL validation.
- [`frontend/`](./frontend/) — 12 skills. React / Next.js / TypeScript and Tailwind + shadcn UI patterns, accessibility, CSS architecture, responsive behavior, bundle performance, and browser-facing security.
- [`platform/`](./platform/) — 8 skills. CI/CD, containers, infrastructure as code, monorepos, release workflows, and developer experience.
- [`security-reliability/`](./security-reliability/) — 14 skills. Threat modeling, secrets/config audits, observability, SLOs, failure modes, and recovery readiness.
- [`testing/`](./testing/) — 9 skills. Unit, integration, E2E, snapshot, coverage, and production-bug regression workflows.
