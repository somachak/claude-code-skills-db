# Data Migration Planner — plain-English guide

## What this skill is for

Plans safe schema and data migrations with backfills, rollout sequencing, rollback paths, and runtime compatibility checks. Use when changing live data models or moving data between stores.

## In simple terms

This skill helps with data migration planner work so you do not have to explain the same rules or workflow every time.

## When you would use it

- migration plan
- backfill
- zero downtime migration
- rollout sequencing
- data change
