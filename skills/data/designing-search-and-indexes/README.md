# Search and Index Design — plain-English guide

## What this skill is for

Designs search schemas, indexing strategies, ranking tradeoffs, and query filters. Use when implementing application search, faceting, vector or lexical retrieval, or hybrid discovery features.

## In simple terms

This skill helps with search and index design work so you do not have to explain the same rules or workflow every time.

## Jargon explained

- `vector search` — A way to search using meaning or similarity rather than only exact keywords.
