# SQL Performance Review — plain-English guide

## What this skill is for

Reviews SQL queries for plan quality, indexing opportunities, cardinality traps, and unnecessary scans. Use when queries slow down APIs, jobs, dashboards, or analytics pipelines.

## In simple terms

This skill helps with sql performance review work so you do not have to explain the same rules or workflow every time.
